# 向量的点乘

![](image/dot1.png)

![](image/dot2.png)

由向量的点乘的意义可以看出, 两个单位向量相乘,

1. 如果点乘越接近 1，代表两个向量之间的夹角越小. 等于 1 时两个向量重合
2. 如果点乘越接近 0，代表两个向量之间的夹角越大，等于 0 时两个向量垂直
